AS-XSTREAM-IPTV

Ativo

Descrição

Aplicativo poderoso e intuitivo projetado para reproduzir listas IPTV no formato XStream. Oferece experiência otimizada para assistir a filmes, séries e canais de TV ao vivo diretamente de listas personalizadas.

Recursos

ASIptvServer.M3U

Beta

Descrição

Biblioteca C# que fornece funcionalidades para ler listas M3U, fazer download de URLs e salvar em arquivo. Separa logo e URL, verifica se é série, filme ou programa de TV.

Funcionalidades

Lê vários formatos de listas, download e salva em arquivo M3U, separa logo e URL, classificação de conteúdo.

Stack

C#, M3U Parser, Regex, HTTP Client

ASIptvServer.Naming

Beta

Descrição

Biblioteca C# que fornece funcionalidades para renomear strings no formato IPTV. Captura o ano, remove parênteses e traços, e organiza o texto de maneira clara e legível.

Funcionalidades

Remove parênteses e traços, verifica se é série de TV, extrai e posiciona ano ao final, organiza texto de forma legível.

Stack

C#, Regex, String Parsing, IPTV
